# 功能

## 账号战略 Agent

| **功能** | **说明** |
| --- | --- |
| 账号定位决策 | 基于行业、目标与资源，确定账号核心定位与差异化方向 |
| 人设定义与固化 | 输出可执行的账号人设（3 句话），统一身份、表达与价值观 |
| 内容结构配比规划 | 决定不同内容类型的比例，避免内容失衡或转化过早 |
| 阶段识别与管理 | 判断账号所处阶段（起号 / 冷启动 / 放量 / 变现），动态调整策略 |
| 目标导向策略切换 | 根据种草、私域或成交目标，切换不同内容与增长策略 |
| 资源约束策略适配 | 结合达人、素材与更新能力，制定现实可执行的账号打法 |
| 更新节奏与频率控制 | 规划发布频率与内容节奏，平衡稳定输出与平台推荐 |
| 风险与禁忌策略 | 输出账号级禁止事项，规避营销号判定与平台风控 |
| 策略一致性校验 | 校验内容与投流行为是否偏离账号战略与人设定位 |
| 战略反馈与修正 | 基于数据反馈持续修正账号定位与阶段策略 |

## 选题策划Agent

| **功能** | **说明** |
| --- | --- |
| 热门话题捕捉 | 自动分析行业内热度上升话题，识别“正在起量”的内容方向，避免追过期热点 |
| 竞品爆款结构拆解 | 对竞品爆款内容进行结构化拆解，抽象出可复用的标题、情绪与内容结构模板 |
| 选题价值评估 | 从共鸣度、竞争度、转化潜力、风险等维度对选题进行打分和优先级排序 |
| 爆款标题生成 | 针对同一选题生成多种爆款标题表达形式，用于 A/B 测试和放量 |
| 爆款结构模板库 | 沉淀不同类型爆款的通用结构（反常识、真相揭露、踩坑经历等）供持续复用 |
| 选题池自动扩展 | 从单一高潜选题裂变出多角度、多人群的内容方向，形成 100+ 选题池 |
| 阶段感知选题策略 | 根据账号所处阶段（冷启动/放量/变现）动态调整选题类型与侧重点 |
| 风险与合规预判 | 在选题阶段提前识别营销风险和平台敏感方向，降低限流与违规概率 |
| 选题执行建议输出 | 为每个高优先级选题给出推荐结构、表达风格与内容侧重点，便于直接执行 |

## 内容生成 Agent

| **功能名** | **说明** |
| --- | --- |
| 结构驱动生文 | 严格按照选题与笔记结构生成正文内容，避免自由发挥导致结构失真 |
| 人设语气生成 | 根据账号 Persona 控制语气、用词与表达习惯，保持内容一致性 |
| 情绪强度调节 | 按账号阶段与内容目标动态调节情绪张力，避免过度煽动 |
| 评论区引导文案生成 | 自动生成非营销式评论引导语，提高互动率 |
| 平台敏感表达规避 | 在生成阶段规避营销、引流、收益承诺等高风险表述 |
| 图文内容协同规划 | 自动规划图文之间的信息分工，避免“图文重复” |
| 生图风格匹配 | 根据账号人设与内容主题生成匹配的小红书风格图片 |
| 场景化图片生成 | 生成人物、物品、使用场景类图片，增强内容代入感 |
| 多图顺序与节奏设计 | 决定首图、过程图、总结图的排列逻辑，提高停留时长 |
| 生图合规与风控 | 避免生成人脸、品牌、广告感强的高风险图片内容 |

## 投流Agent

| **功能** | **说明** |
| --- | --- |
| 投流目标设定 | 根据账号目标（涨粉 / 种草 / 转化）自动选择对应投流策略 |
| 人群定向策略 | 结合行业、人设与历史互动数据，制定精细化投放人群定向方案 |
| 笔记投流候选筛选 | 从已发布笔记中自动筛选最具放量潜力的内容作为投流候选 |
| 投流价值评估 | 从互动率、完读率、评论质量等维度评估笔记的投流性价比 |
| 自动投流决策 | 在预算与风险约束下，自动选择最优笔记执行投流 |
| 投流节奏控制 | 动态控制投流启动时机、强度与时长，避免过度干预自然流量 |

## 内容评估与检验 Agent

| **功能名** | **功能说明** |
| --- | --- |
| 内容质量评估 | 从结构完整性、表达清晰度与信息价值等维度评估内容质量 |
| 人设一致性检验 | 校验内容语气、观点是否符合账号既定人设，避免风格漂移 |
| 平台规则合规检测 | 检测是否存在营销、引流、敏感表达等平台风险 |
| AI 痕迹识别 | 识别过度工整、模板化表达，降低被用户识别为 AI 内容的概率 |
| 情绪与共鸣度评估 | 判断情绪触达是否到位，预测用户共鸣与互动潜力 |
| 投流适配度评估 | 判断内容是否适合进行投流放大，避免低效投放 |
| 图文一致性检验 | 检查图片与文字信息是否协同，避免信息重复或割裂 |
| 发布前评分与建议 | 给出综合评分与可执行修改建议，辅助是否发布决策 |

**Training-Free进化的核心模块**

*   筛选模块，筛选有价值的session，有价值定义：
    
    *   热度：大量用户问过的问题
        
    *   满意度：评估agent认为有问题的回答
        
*   奖励评估
    
    *   自动评估：反思并生成多个候选，评估agent进行推理打分
        
*   语义优势提取
    
    *   基于能打出优势diff的评估，提取明文语义，提取出工作经验
